# Face Recognition Attendance System

A simple Python application that automatically tracks and logs attendance using face detection and recognition.

## Files Included
* `test.py` - The main running script to recognize faces and log attendance.
* `add_faces.py` - Use this script to register and save new faces.
* `faces_data.pkl` & `names.pkl` - Saved facial embeddings and profile labels.
* `haarcascade_frontalface_default.xml` - OpenCV's pre-trained model for detecting faces.
